Chủ đề: Thực hành tính năng RTC của STM32
Chức năng: Dùng đồng hồ thời gian thực trong STM32 để hiển hiện thời gian lên Hercules thông qua UART
Cách sử dụng:
1. Mở file cấu hình (.ioc) để xem cấu hình STM32, Nối các chân UART đúng cấu hình với CP2102.
2. Gắn dây nạp STM32.
3. Mở file project trên STM32CubeIDE để compile và chạy chương trình.
4. Kiểm tra kết nối UART với máy tính và mở Hercules để xem thời gian, lịch trên hercules
